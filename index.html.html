Detailed Hosting Instructions

Important Note

If you're having trouble with GitHub Pages or Netlify, ensure your file is named index.html for automatic loading when you visit the URL.

1

Save the HTML File

Right-click anywhere on this page and select "Save As". Save the file as index.html on your computer.

2

Upload to Hosting Service

Choose either GitHub Pages or Netlify to host your file. Both services are free.

3

Embed in Notion

Create an embed block in Notion and paste your hosted URL. Resize to fit your layout.

GitHub Pages Step-by-Step

Step 1: Create Repository

1. Go to github.com and sign in
2. Create a new repository named instagram-grid
3. Make sure it's set to Public

Step 2: Upload File

1. Click "Add file" > "Upload files"
2. Drag and drop your index.html file
3. Click "Commit changes"

Step 3: Enable Pages

1. Go to Settings > Pages
2. Set Source to "Deploy from a branch"
3. Set Branch to "main" and folder to "/ (root)"
4. Click "Save"

// Your URL will be:
https://<username>.github.io/instagram-grid/

Netlify Step-by-Step

Step 1: Create Account

1. Go to netlify.com
2. Sign up (you can use GitHub account)
3. Complete the signup process

Step 2: Upload Files

1. Go to Sites
2. Drag and drop your index.html file
3. Wait for deployment to complete

Step 3: Get URL

1. Netlify will assign a random URL
2. Copy this URL from the dashboard
3. You can customize it if needed

// Your URL will look like:
https://<random-name>.netlify.app
